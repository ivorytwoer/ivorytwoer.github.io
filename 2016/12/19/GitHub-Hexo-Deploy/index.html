错误记录

  1. Error: spawn Git ENOENT
    不通过cmd而通过git bash或者github的git shell操作，或者将git所在目录加入环境变量
  2. fatal: Not a git repository (or any of the parent directories): .git
    删掉根目录下的文件夹：.deploy_git，重新deploy
